Background

In this mini-practical we are going to use a genome browser to load a genome alignment and identify if the individual has the CYP2C19*2 variant. We are first going to find the co-ordinates for the variant, and then search for the variant in a genome alignment, finally we'll try and identify what genotype the person is at this location and the consequence of the variant.

Step 1: Identify the genomic location of the CYP2C19*2 variant

First we need to find the genomic location of CYP2C19*2 from the Pharmacogene Variation Consortium

Visit the page for CYP2C19*2

Watch this little video which shows how to find the location of the variant

Identifying CYP2C19*2 using a genome browser

We are going to use a simple genome browser called IGV to identify the variant. This genome browser can be accessed online through an online application. This is embedded on this webpage, but can also be accessed here

This is preloaded with some genomic data. This is a file type called a .bam file, which contains the sequencing reads aligned against the reference genome.

Step 2: Load the variant location in IGV

This session of IGV is loaded with one of the platinum Illumina genomes - NA12878 - which is aligned against the GRCh37/hg19 build of the human genome.

Start by navigating to CYP2C19*2 by entering in the location of the variant that you identified in Step 1.

The grey lines represent individual sequence reads. Variant SNPs are shown in a different colour. This genome has been sequenced to 100x coverage, but the number of reads supporting any given base may be lower.

Now we need to use the information shown in IGV to find out whether the variant is heterozygous or homozygous. The short video below demonstrates how to do this.

We can see that our variant is heterozygous, and is 37% of reads are for the reference G, 52% for the variant allele A and one spurious read which has a C. Why do you think we might have one read with a C on it?

Determine the molecular effect of the variant

Now that we have found the variant and determined the genotype we need to ascertain what the effect on the CYP2C19 protein is.

Step 3: Identify the variant effect in IGV

We first need to see what the molecular effect on the codon of the protein is. We can infer this from IGV as demonstrated in the video.

Use the codon chart below to check the effect of the variant, the reference codon is CCA and our variant is CCG.

From this we can see CYP2C19*2 is a synonomous variant, to identify the effect of this variant from first principles we would have to use some other tools outside the scope of this session.

As this is a known variant then for our purposes we need to know where to access information to find out what effect the variant has on the resulting protein. We can use the following online databases to do this: (each link is preloaded to CYP2C19*2)

ClinVar dbSNP OMIM Gnomad 2.1

Make sure you can understand from these datbases what the molecular effect of the variant is.
